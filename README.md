# study-deep-learning-from-scratch

* [book](https://www.oreilly.co.jp/books/9784873117584/)
* [github](https://github.com/oreilly-japan/deep-learning-from-scratch)

## Usage

### Setup

```
$ make build
$ make up
```

### Attach

```
$ make attach
```

### Execute python script

```
$ make python SRC=ch01/1.6.2-sin.py
```
